# `dotfiles`
Collection of my personal dotfiles and project setup scripts.

## How To: Symbolic Links
Symlinks in windows are made using `mklink` (**cmd**).
NOTE: for vscode, the user directory is in `%USERPROFILE%\AppData\Roaming\Code\User\`

## How To: Setup Scripts
* Node: `curl -s https://raw.githubusercontent.com/lap00zza/dotfiles/master/project_setup/node.sh | sh`
